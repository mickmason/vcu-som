How to install frameworks

How to run a bower installation

To use the bower file to run an installation simply run the following command in the command line of the home directory of the project


grunt bower --verbose

You can run this task without the --verbose command, but this allows you to see the output of what bower is doing so you can check that packages are being downloaded properly etc.

Note: This requires grunt to be installed

gruntjs.com/getting-started

Bootstrap

To install Bootstrap into the Project you need to modify bower.json

To modify bower.json just open it with any text editor


In dependencies add the following line of code


"bootstrap-sass" : "latest"

After the ending curly brace of "exportsOverride" : {} add in 


"bootstrap-sass" : {
      "scss" : "lib/*.scss",
      "js" : [
        "js/*.js",
        "dist/js/bootstrap.min.js"
      ],
      "fonts" : "fonts/*"
    },

Now in the command line run the installation using the following command


grunt bower --verbose

How to install Mixin Libraries

Compass

To install Compass you need to open the command line in the directory of your project you wish to install it and run the following command


gem install compass

Note: This requires Ruby to be installed

rubyinstaller.org

Bourbon

Bourbon.io

To install Bourbon you need to open the command line in the directory of your project you wish to install it and run the following command


$ gem install bourbon

Next you need to run the installer to install Bourbon into the directory you are in


bourbon install

This will install the Bourbon library

Note: This requires Ruby to be installed

rubyinstaller.org

How to install Plugins

HTML Validation

To install HTML Validation into the Project you need to modify Gruntfile.js and package.json

To modify these files just open them with any text editor


Gruntfile.js

In Gruntfile.js you will need to use the command Ctrl + F and look for any piece of code commented out containing "validation" or "valid"


package.json

In package.json you will need to add in the following line of code inside of "devDependencies:"


"grunt-html-validation": "latest"

Note: If it is in the list of all the packages, don't forget to add a comma at the end


Finally, you need to create a file called jshintrc.jshintrc in the home directory of the project (Same folder Gruntfile.js is in and add the following code into it


{
  "browser" : true,
  "devel" : true,
  "camelcase" : true,
  "nonbsp" : true,
  "curly" : true,
  "eqeqeq" : true,
  "quotmark" : true,
  "undef" : true,
  "unused" : true
}

Now in the command line run the installation using the following command


npm install
